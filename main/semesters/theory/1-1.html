Тема 1. Знайомство з системним середовищем та початок роботи з C++

1. Мови програмування: класифікація та історія

Історію комп’ютерних наук можна розглядати як історію розвитку інструментів, якими ми спілкуємось з машинами. Початок припадає на XIX ст., коли Чарльз Беббідж розробив механічну обчислювальну машину, а леді Ада Лавлейс написала для неї першу програму.

Визначення:
  • Мова програмування — це штучна мова, створена для розробки програм, які виконуються на комп’ютері.
  • Комп’ютерна програма — це послідовність команд (інструкцій), що забезпечує реалізацію конкретного алгоритму.

Класифікація мов

Сьогодні існує понад 2000 мов програмування. Їх можна поділити за різними ознаками:

  • Низького рівня (Machine-oriented): Максимально наближені до "заліза" комп'ютера (наприклад, Асемблер). Вони швидкі, але складні для людини.
  • Високого рівня (High-level): Зрозумілі людині, використовують слова англійської мови. Саме до таких належать Python, який ви вчили раніше, та C++, який ми починаємо вивчати.

2. Відмінність C++ від Python: Компіляція проти Інтерпретації

Це найважливіший момент переходу. У попередніх класах ви працювали з Python — інтерпретованою мовою.

Як працює Інтерпретатор (Python)?

Він читає вашу програму рядок за рядком і одразу виконує. Якщо в кінці коду є помилка, програма пропрацює до цього моменту і "впаде".

Як працює Компілятор (C++)?

C++ — це компільована мова. Процес створення програми виглядає так:

  1. Написання коду (Source Code): Ви створюєте файл з розширенням .cpp.
  2. Компіляція: Спеціальна програма (компілятор) перекладає весь ваш текст у машинний код. Якщо є хоч одна помилка — файл не створиться.
  3. Лінкування (Linking): До вашого коду додаються системні бібліотеки.
  4. Виконання (Executable): Ви отримуєте окремий файл (наприклад, .exe), який можна запустити навіть без середовища розробки.

Чому C++? Він дає контроль над пам'яттю та високу швидкодію. Саме тому на C++ пишуть ігри, операційні системи (Windows, Linux) та драйвери.

3. Середовище розробки (IDE)

Для написання та компіляції коду нам потрібні інструменти. Ви можете обрати зручний для вас варіант:

Варіант А: Онлайн-компілятори (без встановлення)

Підходять для швидкого старту, роботи з планшета або слабкого ПК.

  • OnlineGDB (має зручний налагоджувач)
  • Ideone
  • Replit (дозволяє зберігати код у хмарі)

Варіант Б: Встановлення на комп'ютер

  • Visual Studio Community: Потужний професійний інструмент (тільки для Windows).
  • Code::Blocks: Легке середовище, працює на старих комп'ютерах.
  • Visual Studio Code: Сучасний редактор (потребує налаштування плагінів C++).

4. Перша програма на C++

На відміну від Python, C++ потребує чіткої структури. Розглянемо класичну програму "Hello, World!".

#include <iostream> // 1. Підключення бібліотеки

using namespace std; // 2. Простір імен

int main() {        // 3. Головна функція
    cout << "Hello, world!" << endl; // 4. Виведення тексту
    return 0;       // 5. Завершення роботи
}

Розбір коду:

  • #include <iostream>: Ми повідомляємо компілятору, що хочемо працювати з введенням та виведенням даних (Input/Output Stream).
  • int main() { ... }: Точка входу. Будь-яка програма на C++ починається з виконання цієї функції.
  • cout: Команда виведення на екран (аналог print).
  • Крапка з комою (;): Обов'язкова в кінці кожної команди! Це найчастіша помилка новачків.

5. Елементарні обчислення та типи даних

C++ має статичну типізацію. Це означає, що ми повинні наперед сказати комп'ютеру, який тип даних будемо зберігати у змінній (ціле число, дробове, текст тощо).

Порівняйте додавання чисел:

Python (динамічна типізація): 
a = 5
b = 10
sum = a + b
print(sum)
C++ (статична типізація): 
int a = 5;      // int = integer (ціле)
int b = 10;
int sum = a + b;
cout << sum << endl;

У C++ ви не можете покласти текст у змінну, яку оголосили як int (число). Це допомагає уникати багатьох помилок ще на етапі написання коду.
