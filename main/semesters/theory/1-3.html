Тема 1.3. Введення даних, простір імен та базові алгоритми

1. Конфлікт імен та простір імен std

Уявіть, що у вашому класі є два учні з ім'ям "Олександр". Якщо вчитель скаже: "Олександре, до дошки!", виникне плутанина (конфлікт імен). Щоб уточнити, вчитель додає прізвище.

У програмуванні теж бувають однакові імена команд. Щоб розрізняти їх, використовують простори імен (namespaces) — це як "прізвища" для команд.

Стандартні команди C++ (такі як cout, cin, endl) знаходяться у просторі імен std (скорочення від standard).

Ми можемо звертатися до них двома способами:

  • Повне ім'я: std::cout (дослівно: "команда cout з родини std"). Це безпечно, але довго писати.
  • Підключення простору: using namespace std;. Ми кажемо комп'ютеру: "Якщо я не уточнюю, вважай, що це з std".

2. Введення даних: cin та cout

До цього ми тільки виводили текст. Тепер навчимося "читати" те, що пише користувач.

Об'єкт Повний запис Напрямок стрілок Пояснення
cout
(Console OUTput)		 std::cout <<
(з програми на екран)		 Виводить дані
cin
(Console INput)		 std::cin >>
(з клавіатури в змінну)		 Зчитує дані

Також важливим є об'єкт endl (End Line) — він переносить курсор на новий рядок (аналог Enter).

int age;
cout << "Скільки тобі років? ";
cin >> age; // Чекає, поки користувач введе число і натисне Enter
cout << "Тобі " << age << " років!" << endl;
Лайфхак: Куди дивляться стрілки, туди течуть дані.
cin >> x (дані летять В змінну x).
cout << x (дані летять ЗІ змінної x на екран).

3. Обробка одиниць виміру (Час, Гроші, Маса)

Комп'ютер зберігає лише числа, він не розуміє, що таке "2 години 5 хвилин". Тому в програмуванні використовують базові одиниці.

  • Для часу — секунди.
  • Для грошей — копійки (центи).
  • Для маси — грами.

Щоб перетворити загальну кількість секунд у формат "Години : Хвилини : Секунди", ми використовуємо ділення / та остачу %.

В 1 годині = 3600 секунд
В 1 хвилині = 60 секунд 
int total_seconds = 3665; 

// Скільки повних годин вміщається?
int hours = total_seconds / 3600; // 1

// Скільки секунд залишилось після того, як забрали години?
int remainder = total_seconds % 3600; // 65

// З решти витягуємо хвилини
int minutes = remainder / 60; // 1

// Те, що лишилось — це секунди
int seconds = remainder % 60; // 5

// Результат: 1 год 1 хв 5 сек

4. Формула Гауса (Сума послідовності)

Існує легенда про малого Карла Фрідріха Гауса. У школі вчитель дав завдання додати всі числа від 1 до 100, щоб зайняти дітей на уроці. Гаус підняв руку через хвилину з правильною відповіддю: 5050.

Він помітив закономірність: пари чисел з країв дають однакову суму.

  • 1 + 100 = 101
  • 2 + 99 = 101
  • ...
Формула суми чисел від 1 до N:
Sum = N * (N + 1) / 2

У програмуванні це дозволяє знайти суму мільйона чисел миттєво, без додавання їх по черзі (без циклів).

long long n; // Використовуємо long long, бо сума може бути великою!
cin >> n;
long long sum = n * (n + 1) / 2;
cout << sum;

5. Кількість чисел на відрізку, кратних k

Часта задача: скільки чисел діляться на k (наприклад, на 5) у діапазоні від A до B?

Для цього не треба перебирати всі числа. Математика дає просту відповідь.

Кількість чисел, кратних k, від 1 до N дорівнює: N / k (цілочисельне ділення).

Тоді для відрізка від A до B формула виглядає так:

Count = (B / k) - ((A - 1) / k)

Логіка: Ми рахуємо, скільки потрібних чисел є до B, і віднімаємо ті, що були до A (які нам не потрібні).
