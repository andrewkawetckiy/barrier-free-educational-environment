Тема 2.1. Умовний оператор та Розгалуження

1. Що таке розгалуження?

У житті ми постійно приймаємо рішення: "Якщо йде дощ, я беру парасольку, інакше — йду в футболці". Це і є розгалуження.

Алгоритм перестає бути прямою лінією і отримує розвилки.

[Image of C++ compilation process diagram]

2. Головна відмінність: Python vs C++

Ви вже знайомі з оператором if у Python. У C++ логіка та сама, але запис суворіший.

Золоте правило C++: Умова завжди в круглих дужках (...), а дії — у фігурних {...}.
Python (Як ви звикли) C++ (Як треба тепер) 
if x > 0:
    print("Positive")
else:
    print("Negative")
  • Без дужок біля умови
  • Двокрапка : в кінці
  • Відступи (Tab) обов'язкові
 
if (x > 0) {
    cout << "Positive";
} else {
    cout << "Negative";
}
  • Дужки () обов'язкові
  • Фігурні дужки {} замість відступів
  • Крапка з комою ; в кінці команд

3. Логічні вирази (Comparison & Logic)

Як сказати комп'ютеру "Якщо x більше 5 І менше 10"?

Оператори порівняння (так само, як в Python):

  • == — дорівнює (не плутати з =)
  • != — не дорівнює
  • >, <, >=, <= — більше/менше

Логічні зв'язки (Відрізняються!):

Дія Python C++ Приклад C++
І (AND) and && if (age > 10 && age < 18)
АБО (OR) or || if (day == 6 || day == 7)
НЕ (NOT) not ! if (!isReady)

4. Типи розгалужень

4.1. Неповне розгалуження (Тільки if)

Робимо щось тільки якщо умова істинна. Якщо ні — просто йдемо далі.

if (money >= 100) {
    cout << "Купуємо піцу!";
}
// Якщо грошей менше, просто мовчимо і йдемо далі

4.2. Повне розгалуження (if ... else)

Або одне, або інше. Третього не дано.

if (number % 2 == 0) {
    cout << "Число парне";
} else {
    cout << "Число непарне";
}

4.3. Багатоальтернативне (if ... else if ... else)

Коли варіантів багато (як у прикладі з підручника про результати матчів).

if (place == 1) {
    cout << "Золото";
} else if (place == 2) {
    cout << "Срібло";
} else if (place == 3) {
    cout << "Бронза";
} else {
    cout << "Учасник";
}

5. Базові алгоритми з числами

5.1. Максимальне з двох

Найпростіший спосіб знайти максимум:

int max_num;
if (a > b) {
    max_num = a;
} else {
    max_num = b;
}

Або скорочено через вбудовану функцію: max_num = max(a, b); (потрібна бібліотека <algorithm>).

5.2. Сортування трьох чисел

Задача: Є числа a, b, c. Вивести їх у порядку зростання.

Алгоритм "бульбашки" (спрощений): порівнюємо пари і міняємо місцями, якщо вони стоять неправильно.

// Якщо a більше b, міняємо їх місцями (swap)
if (a > b) swap(a, b);
// Тепер у a точно менше число, ніж у b.
// Перевіряємо b і c...
if (b > c) swap(b, c);
// Тепер найбільше число точно в кінці (в c).
// Але a і b могли знову стати неправильно, перевіряємо ще раз:
if (a > b) swap(a, b);

cout << a << " " << b << " " << c;
