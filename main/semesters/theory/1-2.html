Тема 1.2. Структура програм, змінні та цілі числа

1. Структура програми та Коментарі

У попередній темі ми вже бачили скелет програми. Давайте розглянемо його детальніше та навчимося залишати нотатки для себе — коментарі.

Коментарі ігноруються комп'ютером. Вони потрібні лише програмісту, щоб пояснити складний код.

Види коментарів у C++:
  • // — Однорядковий коментар. Все, що після двох скісних рисок у цьому рядку, комп'ютер не бачить.
  • /* ... */ — Багаторядковий коментар. Можна писати цілі поеми всередині зірочок.
#include <iostream> 

// Це головна функція. З неї все стартує.
int main() {
    /* Ця команда виводить текст
       на екран монітора */
    std::cout << "Привіт!"; 

    return 0; // Завершення програми
}

Форматування коду

C++ дозволяє писати весь код в один рядок, але читати це неможливо. Хороший стиль ("Good style") — це:

  • Кожна команда з нового рядка.
  • Відступи (Tab) всередині фігурних дужок { }.
  • Пробіли навколо знаків = та + для читабельності.

2. Змінні: Ініціалізація та Присвоювання

В Python змінна була як наліпка, яку можна приклеїти до чого завгодно. У C++ змінна — це коробка певного розміру.

int
age
16
← Ім'я на коробці, тип (розмір) і вміст.

Щоб створити змінну в C++, треба пройти три кроки:

  1. Оголошення (Declaration): Ми кажемо комп'ютеру: "Виділи пам'ять під коробку типу int і назви її x".
  2. Ініціалізація (Initialization): Ми кладемо туди перше значення.
  3. Присвоювання (Assignment): Ми змінюємо значення в коробці.
int x;        // Оголошення (в коробці зараз сміття!)
x = 10;       // Присвоювання
int y = 20;   // Оголошення + Ініціалізація (Найкращий варіант)

3. Цілочисельні типи даних (Integer types)

Чому в C++ так багато типів для чисел? Щоб економити пам'ять. Якщо вам треба зберегти вік людини (до 120 років), не варто брати "величезну коробку", яка може вмістити мільярди.

Тип C++ Приблизний діапазон Для чого використовують
short від -32,768 до 32,767 Невеликі числа (рік народження, температура)
int від -2 мільярдів до +2 мільярдів Основний тип. Використовуйте його за замовчуванням.
long long Дуже великі числа (±9×10¹⁸) Гроші світу, відстань до зірок, ID користувачів.

4. Математика та Оператори

Тут криється головна пастка для тих, хто переходить з Python. Уважно розгляньте таблицю операторів.

Важливо! У C++ оператор / працює по-різному для цілих і дробових чисел.
5 / 2 = 2 (ціле ділення, дробова частина відкидається).

Арифметичні оператори

Оператор Назва Опис та приклад
+ Додавання 10 + 5 → результат 15
- Віднімання 10 - 5 → результат 5
* Множення 10 * 5 → результат 50
/ Цілочисельне ділення Якщо обидва числа цілі, результат теж цілий.
10 / 3 → результат 3 (не 3.33!)
% Остача від ділення Повертає те, що не поділилося.
10 % 3 → результат 1 (бо 10 = 3*3 + 1)

Скорочений запис (для лінивих програмістів)

Програмісти не люблять писати зайве. Тому існують скорочені оператори:

Скорочений запис Повний аналог Приклад (якщо x = 10)
x += 5; x = x + 5; x стане 15
x -= 3; x = x - 3; x стане 7
x *= 2; x = x * 2; x стане 20
x++; x = x + 1; Збільшити на 1 (Інкремент)

Приклад використання остачі (%)

Як розбити число 123 на цифри? Це класична алгоритмічна задача.

  • 123 % 10 → отримаємо 3 (остання цифра).
  • 123 / 10 → отримаємо 12 (відрізали останню цифру).
  • 12 % 10 → отримаємо 2.
