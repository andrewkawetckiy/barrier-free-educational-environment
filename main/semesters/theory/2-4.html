Тема 2.4. Системи числення

1. Навіщо це нам?

Люди рахують десятками (бо 10 пальців). Комп'ютери — двійками (є струм / немає струму). Програмісти часто використовують шістнадцяткову систему (HEX), щоб скорочувати довгі двійкові коди (наприклад, коди кольорів #FF5733).

Десяткова (DEC) Двійкова (BIN) Шістнадцяткова (HEX)
000000
101010A
151111F
25511111111FF

2. Маніпулятори cout (HEX, OCT)

У C++ перемикати системи числення при виводі дуже просто. Достатньо "кинути" в cout спеціальну команду.

#include <iostream>
using namespace std;

int main() {
    int num = 255;
    
    cout << hex << num << endl; // Виведе "ff"
    cout << uppercase << num << endl; // Виведе "FF"
    cout << dec << num << endl; // Повернули назад у десяткову -> 255
    
    return 0;
}
Увага: Маніпулятори "липучі". Якщо ви один раз написали cout << hex, всі наступні числа будуть у HEX, поки ви не напишете cout << dec.

3. Як вивести у Двійковій (BIN)?

На жаль, у C++ немає стандартного cout << bin. Але є бібліотека <bitset>.

Вона дозволяє показати число як набір бітів.

#include <iostream>
#include <bitset> // Підключаємо бібліотеку

using namespace std;

int main() {
    int x = 10;
    // <8> означає показати 8 бітів
    cout << bitset<8>(x) << endl; // Виведе 00001010
    return 0;
}

4. Алгоритм переведення (Математика)

Якщо нам не можна використовувати готові бібліотеки, ми ділимо число на основу системи (на 2 або на 16) і записуємо остачі задом наперед.

Приклад: 13 у двійкову
13 / 2 = 6 (остача 1)
6 / 2 = 3 (остача 0)
3 / 2 = 1 (остача 1)
1 / 2 = 0 (остача 1)
Читаємо знизу вгору: 1101
